Classical Bit

An abstract two-state classical system is any system that can be in one of two possible abstract states. Let {A, B} be the set of possible abstract states. i.e. The system is either in state A or in state B. An abstract two-state system can be implemented in different physical media as long as the physical medium can represent the two abstract states A and B. To implement an abstract two-state system (with abstract states A and B) in a physical medium, we use a property of the medium that can have two possible values. One value will represent the abstract state A while the other value will represent abstract state B. The table below shows some physical media and the physical properties and values used to represent the two abstract states A and B.

Physical Medium Physical Property State A State B
Punched Card Existence of a Hole No Hole Has a Hole
Electronic Circuit Voltage Level Low (≈ 0V) High (≈ 5V)
Flash Memory Cell Cell Charge ‘High’ Charge No or ‘Low’ Charge
Magnetic Drive Cell Magnetic Polarity South-North North-South

The bit, the most basic unit of information in computing, is the amount of information you gain if the actual state of a two-state system is revealed to you (when the system has an equal chance of being in state A or being in state B). It is the same amount of information you gain when you get an answer to a yes/no question when there is an equal chance of getting a ’yes’ or getting a ’no’ answer. The term ’bit’ stands for "binary digit" or "binary information digit". A bit can either be the digit ’0’ or the digit ’1’. When using the bit to represent the states of a two-state system, you use the ’0’ bit to represent one state and the ’1’ bit to represent the other state. It has been the convention to use the bit to represent states of a two-state system. On the table above, it is often the case that state A is represented by the ’0’ bit while state B is represented by the ’1’ bit.

A bit string is simply a string of bits. e.g. 1, 01, 111, 0001. Bit string are used to encode information and the amount of information a bit string holds is the number of bits it has. The number of bits a bit string has is its length. i.e. The string 10010 is bit string with 5 bits of information. It is a bit string of length 5. Most digital data today like pictures, audio, video, and files are stored as bit strings (aka binary strings). In order to represent a bit string, you need multiple two-state systems with each system holding one bit of information. For example, a magnetic drive with a 1-gigabyte (1GB) capacity stores a bit string of length 8 billion ( 1 byte = 8 bits, 1 gigabyte = 10003 bytes) so you need 8 billion two-state systems and for magnetic drives these two-state systems are magnetic drive cells where each cell is a ’magnetic domain’ with a specific magnetic orientation that represent either 0 or 1.

The set of symbols Σ = {0, 1} is called the alphabet of bit strings. The set Σ* (pronounced as "Sigma star") is the set of all bit strings (of any lenth). i.e. Σ* = {ε, 0, 1, 00, 01, 10, 11, 000, 001, ...}. The ε (or sometimes λ) symbol is used for the “empty string", the bit string of length 0. The set Σ+ (pronounced as "Sigma plus") is the set of all bitstring excluding the empty string ε. The set Σn is the set of bit strings of length n. i.e. Σ2 = {00, 01, 10, 11}. The concatenation operation is a binary operation that strings two bit strings together. i.e. Let B1 and B2 be bits strings, the the concatentation of B1 with B2, denoted by B1|B2, is simply B1B2. e.g. If B1 = 111 and B2 = 00100, then B1|B2 = B1B2 = 11100100.

Quantum Bit

In a two-state quantum system, similar to a two-state classical system, if you observe the system you will ‘see’ it in either one of two definite states. Let $\{\ket{A}, \ket{B}\}$ be the set of observable states. i.e. The system can be observed to be either in state $\ket{A}$ or in state $\ket{B}$.

An abstract two-state system can be implemented in different physical media as long as the physical medium can represent the two abstract states A and B. To implement an abstract two-state system (with abstract states A and B) in a physical medium, we use a property of the medium that can have two possible values. One value will represent the abstract state A while the other value will represent abstract state B. The table below shows some physical media and the physical properties and values used to represent the two abstract states A and B.

Physical Medium Physical Property State $\ket{A}$ State $\ket{B}$
Electron Electron Spin Spin Up Spin Down
Atomic Nucleaus Nuclear Spin Spin Up Spin Down
Light Polarity Horizontal Vertical

Superposition


$$\ket{\psi} = \alpha \ket{0} + \beta \ket{1}$$

Algebraic representation of the quantum state $\ket{\psi}$ using the basis $\{\ket{0}, \ket{1}\}$:

In a two-level system with basis states $\{\ket{0}, \ket{1}\}$, by observing the system you will either see that system in state $\ket{0}$ or in state $\ket{1}$. The state $\ket{\psi} = \alpha \ket{0} + \beta \ket{1}$ is not an actual state that you can observe but it gives you information about the probability of seeing the system in state $\ket{0}$ and the probability of seeing the system in state $\ket{1}$ if you observe the system. The α in $\ket{\psi}$ is called the amplitude of the basis state $\ket{0}$ while the β in $\ket{\psi}$ is amplitude of basis state $\ket{1}$. The amplitudes associated with the basis state tells you about the probability associated with seeing the particular basis state if observe the system. Getting the square of the amplitude of the a basis state (technically, getting the square modulus) will give you the probability of seeing that basis state if you observe the system. i.e. |α|2 is the probability of seeing the basis state $\ket{0}$ while |β|2 is the probability of seeing the basis $\ket{1}$.

Born Rule
|α|2 + |β|2 = 1

Tensor product

Let the first qubit be $\ket{q_1}$:


$$\ket{q_1} = \alpha \ket{0} + \beta \ket{1} = \begin{bmatrix} \alpha \\ \beta \end{bmatrix}$$

Let the second qubit be $\ket{q_2}$:


$$\ket{q_2} = \gamma \ket{0} + \delta \ket{1} = \begin{bmatrix} \gamma \\ \delta \end{bmatrix}$$

The tensor product of $\ket{q_1}$ and $\ket{q_2}$, denoted by $\ket{q_1} \otimes \ket{q_2}$ (or simply $\ket{q_1q_2}$), is computed as follows:
$$\begin{aligned} \ket{q_1} \otimes \ket{q_2} &= \ket{q_1q_2} \\ &= (\alpha \ket{0} + \beta \ket{1}) \otimes (\gamma \ket{0} + \delta\ket{1}) \\ &= [(\alpha \ket{0}) \otimes (\gamma \ket{0} + \delta \ket{1})] + [(\beta \ket{1}) \otimes (\gamma \ket{0} + \delta \ket{1})] \\ &= [\alpha \gamma \ket{00} + \alpha \delta \ket{01}] + [\beta \gamma \ket{10} + \beta \delta \ket{11}] \\ &= \alpha \gamma \ket{00} + \alpha \delta \ket{01} + \beta \gamma \ket{10} + \beta \delta \ket{11} \\ \end{aligned}$$

As column vectors:
$$\begin{aligned} \ket{q_1q_2} &= \ket{q_1} \otimes \ket{q_2} \\ &= \begin{bmatrix} \alpha \\ \beta \end{bmatrix} \otimes \begin{bmatrix} \gamma \\ \delta \end{bmatrix} \\ &= \begin{bmatrix} \alpha \begin{bmatrix} \gamma \\ \delta \end{bmatrix} \\ \beta \begin{bmatrix} \gamma \\ \delta \end{bmatrix} \end{bmatrix} = \begin{bmatrix} \alpha \cdot \gamma \\ \alpha \cdot \delta \\ \beta \cdot \gamma \\ \beta \cdot \delta \end{bmatrix} \\ &= \begin{bmatrix} \alpha \gamma \\ \alpha \delta \\ \beta \gamma \\ \beta \delta \end{bmatrix}\end{aligned}$$
